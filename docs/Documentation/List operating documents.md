# **ВЕДОМОСТЬ ЭКСПЛУАТАЦИОННЫХ ДОКУМЕНТОВ**
## **Программный комплекс "Умный Дом" (Smart Home System)**

| № п/п | Обозначение документа | Наименование документа | Примечание |
|-------|----------------------|------------------------|------------|
| 1 | SHS-ЭД-01 | **Техническое описание** | Содержит общие сведения о системе, назначение, условия применения, состав системы, описание функционирования. |
| 2 | SHS-ЭД-02 | **Руководство системного администратора** | Описывает процедуры установки, настройки, обновления, резервного копирования и администрирования системы. |
| 3 | SHS-ЭД-03 | **Руководство оператора** | Содержит подробное описание пользовательского интерфейса, последовательности действий для выполнения всех функций системы. |
| 4 | SHS-ЭД-04 | **Программа и методика испытаний** | Описывает методы, средства и порядок проведения приемо-сдаточных и периодических испытаний системы. |
| 5 | SHS-ЭД-05 | **Формуляр** | Содержит паспортные данные комплекса, сведения о комплектности, гарантийные обязательства, учет использования и ремонтов. |
| 6 | SHS-ЭД-06 | **Описание применения** | Прикладные инструкции по использованию системы для решения типовых задач (сценарии). |
| 7 | SHS-ЭД-07 | **Акт приема-сдачи системы в эксплуатацию** | Документ, подтверждающий соответствие системы требованиям и готовность к эксплуатации. |
| 8 | SHS-ЭД-08 | **Журнал эксплуатации системы** | Для регистрации текущих операций, сбоев, изменений конфигурации и обращений в поддержку. |
| 9 | SHS-ЭД-09 | **Альбом экранных форм** | Визуальное представление всех интерфейсных окон, диалогов и элементов управления системы. |
| 10 | SHS-ЭД-10 | **Справочное руководство (онлайн-справка)** | Контекстная справка, интегрированная в графический интерфейс системы. |
| 11 | SHS-ЭД-11 | **Инструкция по безопасности** | Меры по обеспечению информационной, программной и технической безопасности при эксплуатации. |
| 12 | SHS-ЭД-12 | **Регламент технического обслуживания** | План-график профилактических работ, проверок и обслуживания системы. |
| 13 | SHS-ЭД-13 | **Методика восстановления после сбоя** | Пошаговые инструкции по восстановлению работоспособности системы в аварийных ситуациях. |
| 14 | SHS-ЭД-14 | **Каталог кодов ошибок и рекомендации** | Перечень возможных ошибок системы, их причины и способы устранения. |
| 15 | SHS-ЭД-15 | **Лист регистрации изменений** | Учет изменений в документации при обновлениях и модификациях системы. |

---

### **Примечания:**

1. Все документы разрабатываются в электронном виде (форматы: PDF, DOCX, HTML).
2. Документы SHS-ЭД-03 (Руководство оператора) и SHS-ЭД-10 (Справочное руководство) поставляются в составе дистрибутива системы и доступны из главного меню.
3. Документы SHS-ЭД-02, SHS-ЭД-04, SHS-ЭД-05, SHS-ЭД-11, SHS-ЭД-12, SHS-ЭД-13 предоставляются заказчику на этапе ввода системы в эксплуатацию.
4. Документы SHS-ЭД-07, SHS-ЭД-08 являются эксплуатационными и заполняются в процессе использования системы.
