# **Методика испытаний системы "Умный Дом"**

Настоящий документ описывает методы проверки выполнения функциональных требований программного комплекса "Умный Дом". Каждая проверка выполняется путём воспроизведения последовательности действий оператора, описанных в **Руководстве оператора**. Результаты проверки фиксируются в Протоколе испытаний.

---

## **1. Методы проверки выполнения функции инициализации системы**
Проверка выполнения указанной функции выполняется согласно **п. 3.1 "Запуск системы"** Руководства оператора.  
Проверка считается завершённой в случае успешного запуска всех сервисов, отображения главного окна GUI и корректной инициализации устройств. По результатам проверки в Протокол испытаний вносится запись: *«п. 3.1 выполнен»*.
